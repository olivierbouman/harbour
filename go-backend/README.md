# back-end in Go

### LOCAL SETUP

# Add path to go storage location

run the following command from the location where you want to use Go: `export PATH=$PATH:/usr/local/go/bin`

# Add go module file

go mod init go-backend

# Download all the used Go modules

go mod tidy

# Run go-backend

go run cmd/web/\*.go
